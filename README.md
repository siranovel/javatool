javatool
========
c言語からJNIライブラリを使用したツールです。

## Description ##
### ツールの種類 ###
* dspfields   
  javaクラスのフィールド一覧を表示するプログラム
* dspjarfileinfo  
  jarファイルの情報を表示するツールです。
* dspmethods  
  javaクラスの情報を表示するプログラム
* dsppropinfo  
  javaのシステムプロパティ情報を表示するプログラム
*  dspScriptEngine  
  javaのScript Engine情報を表示するプログラム
