## Usage ##
* dspfields  
  $ dspfields フルクラス名
* dspjarfileinfo  
  $ dspjarfileinfo jarファイル名
* dspmethods  
  $ dspmethods フルクラス名
* dsppropinfo  
  $ dsppropinfo
* dspScriptEngine  
  $ dspScriptEngine
* dspmodinfo  
  $ dspmodinfo フルクラス名
* dspjvmflags  
  $ dspjvmflags

## install ##
    git clone https://github.com/siranovel/javatool.git  
    cd javatool  
    rpm -ivh <rpmファイル名>
